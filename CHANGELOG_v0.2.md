# AJ Smart Indicator — Changelog

## v0.2 — 18 July 2026

- First implementation of the original AJ Smart Indicator.
- Added trend engine using a configurable EMA alignment filter.
- Added confirmed-pivot swing engine.
- Added ATR engine and strong-candle quality filter.
- Added hybrid stop loss: compares ATR and confirmed swing structure.
- Added Entry, Stop Loss, Target 1, and Target 2 level plotting.
- Added confirmed bullish and bearish setup markers and alert conditions.
- All setup and level updates are gated to completed candles.
